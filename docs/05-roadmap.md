# Roadmap

## MVP — v1.0 (Atual)

**Objetivo:** Validar o produto com primeiros clientes pagantes

| Funcionalidade | Status |
|---|---|
| Autenticação (login/registro) | ✅ |
| Chat com IA (web) | ✅ |
| Agente Financeiro | ✅ |
| Agente Estoque | ✅ |
| Agente Clientes | ✅ |
| Dashboard básico | ✅ |
| API REST completa | ✅ |
| Docker / infra local | ✅ |
| Integração WhatsApp | 🔧 Em desenvolvimento |

---

## v1.1 — Cobranças e Alertas

- Envio automático de cobranças por WhatsApp
- Alertas de estoque baixo via notificação
- Relatórios em PDF
- Confirmação de agendamentos

## v1.2 — Relatórios e BI

- Dashboard com gráficos (Recharts)
- Relatório de DRE simplificado
- Fluxo de caixa projetado
- Análise de inadimplência
- Previsão financeira com IA

## v1.3 — Integrações

- Integração com bancos (Open Finance)
- Emissão de NF-e (API SEFAZ)
- Integração com sistemas de pagamento (Pix, boleto)
- Importação de extrato bancário OFX

## v2.0 — Expansão

- Aplicativo mobile (React Native)
- Agente de RH (funcionários, ponto, folha)
- Agente de Compras (pedidos a fornecedores)
- Multi-idioma (inglês, espanhol)
- Marketplace de integrações

---

| NPS | > 50 | > 70 |
| Tempo médio de resposta | < 5s | < 3s |
